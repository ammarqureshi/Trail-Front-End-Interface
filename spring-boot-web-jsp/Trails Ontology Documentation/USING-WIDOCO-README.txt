To generate documentation for the ontology:

1. Download the widoco-X.X.X-jar-with-dependencies.jar from https://github.com/dgarijo/WIDOCO/releases/tag/v1.4.7

2. Run the jar file and choose to "Create template documentation from ontology file" selecting the ontology Turtle file. You can rename the documentation output folder from myDocumentation here as well. Click next.

3. Click next at step 2

4. For step 3, tick "Include Annotation Properties" and untick "Include Named Individuals", then click generate.

5. Click finish.

6. Open <documentationfolder>/doc/index-en.html to open the documentation
